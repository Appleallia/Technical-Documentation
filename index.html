What is HTML?

HTML stands for HyperText Markup Language. Its a standard a language used to for creating Web pages , invented by Tim Berners-Lee in the 1989. It is now widely used as building block for website. HTML uses tags to provide information such as headings, paragraphs, images, and links.

What is Cascading Style Sheet?

CSS is used to control the appearance of HTML elements, including colors, fonts, spacing, and layout. This is often added to enhance the creativity of an HTML document. Which can be done three methods , such as internal , external and inline styles.

What is JavaScript?

JavaScript is a programming language that adds interactivity and dynamic behavior to webpages.It makes your website more engaging and useful.

How to Create a Survey Form

HTML forms are created using the <form> element along with input fields such as text boxes, radio buttons, and submit buttons.
