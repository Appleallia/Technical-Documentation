What is HTML?

HTML, which stands for HyperText Markup Language, is the standard language used to create and structure webpages. It was developed by Tim Berners-Lee in 1989 and is considered the foundation of the World Wide Web. Every website, regardless of how advanced or interactive it may appear, begins with HTML.

HTML works by using elements known as tags to define different types of content. These tags tell the browser how to display information such as headings, paragraphs, images, links, tables, and forms. For example, the <h1> tag is used for main headings, while the <p> tag is used for paragraphs of text.

One of the main strengths of HTML is its simplicity and structure. It does not control how a webpage looks or behaves, but instead focuses on organizing content in a meaningful way. This allows other technologies, such as CSS and JavaScript, to work alongside HTML to enhance design and interactivity.

HTML is essential for accessibility and search engine optimization (SEO) because it helps screen readers and search engines understand the content of a webpage. Without proper HTML structure, websites would be difficult to navigate and interpret.

A real-world example of HTML in use can be seen on educational websites such as W3Schools HTML Tutorial, which demonstrates how HTML is used to build webpages from the ground up.

What is Cascading Style Sheet?

Cascading Style Sheets (CSS) is a language used to control the visual appearance of HTML elements on a webpage. While HTML focuses on structure and content, CSS is responsible for design elements such as colors, fonts, spacing, layouts, and positioning.

CSS is used to improve the user experience by making webpages visually appealing and easier to read. Without CSS, websites would appear plain and unorganized, using default browser styles. CSS allows developers to maintain consistency across multiple pages and make changes efficiently.

CSS Rules and Syntax

A CSS rule consists of a selector and a declaration block. The selector identifies the HTML element being styled, while the declaration defines how it should appear. 

p {
  color: blue;
  font-size: 16px;
}

Types of CSS

  • Inline CSS: Applied directly within an HTML tag.
  • Internal CSS: Written inside a <style> tag within the HTML document. Which is what I used for this project.
  • External CSS: Stored in a separate .css file and linked to multiple webpages.

Example of inline CSS: 

<p style="color:red;">This text is red</p>

CSS plays a critical role in modern web development by separating content from design, making websites easier to maintain and scale.

What is JavaScript?

JavaScript is a programming language used to add interactivity and dynamic behavior to webpages. Unlike HTML and CSS, which focus on structure and appearance, JavaScript allows websites to respond to user actions such as clicks, form submissions, and keyboard input.

JavaScript is commonly used to validate forms, display alerts, update content without reloading the page, and create interactive features such as dropdown menus and pop-ups.

Ways to Add JavaScript to HTML

1. Inline JavaScript

<button onclick="alert('Hello!')">Click Me</button>

2. Internal JavaScript

<script>
  function greet() {
    alert("Welcome to my webpage!");
  }
</script>

3. External JavaScript

<script src="script.js"></script>

External JavaScript is the preferred method for larger projects because it keeps code organized and allows scripts to be reused across multiple webpages.
