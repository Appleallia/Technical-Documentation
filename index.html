What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create webpages and was invented by Tim Berners-Lee in 1989.

HTML uses elements called tags to structure content such as headings, paragraphs, images, links, and forms. It is the foundation of all websites.

Example of a real HTML-based website: W3Schools HTML Tutorial

What is Cascading Style Sheet?

Cascading Style Sheets (CSS) control the visual appearance of HTML elements, including layout, colors, fonts, and spacing.

CSS can be applied using inline styles, internal style sheets, or external style sheets.

What is JavaScript?

JavaScript is a programming language that adds interactivity and dynamic behavior to webpages. It allows websites to respond to user actions.

How to Create a Survey Form

Step-by-Step Guide

  1. Create a form using the <form> tag.
  2. Use labels to describe input fields.
  3. Add inputs for name, email, and age.
  4. Use radio buttons for selections.
