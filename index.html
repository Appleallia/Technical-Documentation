What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web using elements such as headings, paragraphs, images, and links.

What is Cascading Style Sheet?

CSS is used to control the appearance of HTML elements, including colors, fonts, spacing, and layout.

What is JavaScript?

JavaScript is a programming language that adds interactivity and dynamic behavior to webpages.

How to Create a Survey Form

HTML forms are created using the <form> element along with input fields such as text boxes, radio buttons, and submit buttons.
